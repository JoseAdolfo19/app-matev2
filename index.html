Bórrala de la lista de dependencias. --- ### **PASO 2: Agregar la librería nueva** En el mismo archivo `package.json`, en la sección de "dependencies", agrega esta línea: ``` "@react-oauth/google": "^0.11.0" ``` --- ### **PASO 3: Guardar el archivo** Guarda los cambios en `package.json` (Ctrl + S). --- ### **PASO 4: Eliminar carpetas viejas** Abre tu terminal (PowerShell) y ejecuta estos comandos uno por uno: ``` cd C:\xampp\htdocs\app-matev2\frontend Remove-Item -Recurse -Force node_modules Remove-Item -Force package-lock.json ``` Esto elimina las carpetas donde están instaladas todas las librerías para que puedas instalar todo desde cero. --- ### **PASO 5: Instalar las dependencias (con la solución rápida)** Ejecuta este comando en la terminal: ``` npm install --legacy-peer-deps ``` El `--legacy-peer-deps` le dice a npm que ignore los conflictos de versiones y que instale todo como pueda. --- ### **PASO 6: Ejecutar el proyecto** Una vez que termine la instalación, ejecuta: ``` npm run dev ``` --- ## 📝 **Resumen de lo que debes hacer (lista de tareas)** - [ ] **Abrir** el archivo `frontend/package.json` - [ ] **Buscar** la línea que dice `"react-google-login"` - [ ] **Eliminar** esa línea completamente - [ ] **Agregar** la línea `"@react-oauth/google": "^0.11.0"` - [ ] **Guardar** el archivo - [ ] **Abrir** la terminal en la carpeta `frontend` - [ ] **Ejecutar** `Remove-Item -Recurse -Force node_modules` - [ ] **Ejecutar** `Remove-Item -Force package-lock.json` - [ ] **Ejecutar** `npm install --legacy-peer-deps` - [ ] **Ejecutar** `npm run dev` --- ## ⚠️ **Y después de instalar...** El botón de "Iniciar sesión con Google" va a cambiar un poco porque la nueva librería funciona diferente.
